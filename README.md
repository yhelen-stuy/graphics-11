# graphics-engine

Graphics engine in Golang for Graphics high school senior year class

## How to Run

To compile to an executable, run

```
$ make
```

After compiling, to parse through a script and run the graphics engine, run:

```
$ ./main FILENAME
```

For example

```
$ ./main lighting.mdl
```
